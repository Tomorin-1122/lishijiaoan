# 教案工坊 — 教学设计生成器

一个基于浏览器的教学设计/教案生成工具，帮助教师快速创建专业的教学文档。

## 📖 项目简介

**教案工坊** 是一个单页 Web 应用（SPA），专为教育工作者设计，提供直观、高效的教案编写体验。通过简洁的界面和智能化的功能，让教学设计变得更加轻松。

## ✨ 主要功能

- **📝 多模块教案编辑** - 支持教学目标、教学重难点、教学过程、板书设计等多个教学环节
- **🎨 可视化界面** - 采用优雅的配色方案和清晰的布局，提供舒适的编辑体验
- **📄 PDF 导出** - 一键将教案导出为 PDF 格式，方便打印和分享
- **💾 本地保存** - 支持将教案保存到本地，随时继续编辑
- **🔄 模块化设计** - 可灵活添加、删除、调整教学环节模块
- **🖼️ 图片插入** - 支持在教学设计中插入图片素材
- **🏷️ 快捷标签** - 提供常用教学术语快捷插入功能

## 🛠️ 技术栈

- **HTML5** - 语义化结构
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Vanilla JavaScript** - 原生 JavaScript 实现交互逻辑
- **html2pdf.js** - PDF 生成库
- **Font Awesome** - 图标库
- **Google Fonts** - 思源宋体/思源黑体中文字体

## 🚀 使用方法

### 直接打开

直接在浏览器中打开 `index.html` 文件即可使用：

```bash
# macOS / Linux
open index.html

# Windows
start index.html
```

### 使用本地服务器（推荐）

为了获得最佳体验，建议使用本地服务器：

```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js 的 http-server
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 💡 使用提示

1. **左侧导航栏** - 快速切换不同功能区域
2. **表单填写** - 依次填写课题、教材、教学目标等信息
3. **模块管理** - 点击「+」添加新模块，点击删除按钮移除模块
4. **导出教案** - 编辑完成后点击「导出 PDF」生成文档
5. **保存进度** - 定期点击「保存」防止数据丢失

## 🎨 界面特色

- **古典雅致配色** - 采用传统纸张色系，营造温馨的备课氛围
- **清晰层级结构** - 侧边栏 + 主内容区的经典布局
- **响应式交互** - 流畅的动画过渡和视觉反馈

## 📝 注意事项

- 本应用完全在浏览器端运行，数据保存在本地
- 建议使用现代浏览器（Chrome、Edge、Firefox、Safari）
- 导出 PDF 功能需要网络连接以加载 html2pdf.js 库
